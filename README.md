# 디지털 가족 앨범 (Digital Family Album)

## 프로젝트 목표

비싼 디지털 앨범을 구매하는 대신, 사용하지 않는 오래된 태블릿을 활용하여 웹 기반의 디지털 가족 앨범을 만드는 것을 목표로 합니다.

## 프로젝트 계기

시중의 전자 앨범 가격이 부담스러워, 직접 만들어 보기로 했습니다. 집에 방치되어 있는 구형 태블릿들은 웹 브라우저 구동이 가능하기 때문에, 이를 활용하여 추억을 담는 앨범으로 재탄생시키고자 합니다.

## 개발 계획

이 프로젝트는 여러 단계에 걸쳐 개발될 예정입니다.

### 1단계: 프론트엔드 개발
- 원하는 디자인의 앨범 UI를 구현합니다.
- HTML, CSS, JavaScript를 사용하여 정적인 웹 페이지를 구성합니다.

### 2단계: 백엔드 개발
- 사진을 업로드하고 관리할 수 있는 백엔드 서버를 구축합니다.
- 서버를 통해 사진을 추가하면 앨범에 동적으로 반영되도록 합니다.

### 3단계: 사용자 시스템 도입
- 데이터베이스를 연동하여 사용자 로그인 및 회원가입 기능을 구현합니다.
- 여러 사용자가 각자의 앨범을 만들고 관리할 수 있는 플랫폼으로 확장합니다.
